# ClinicFlowAC + OVWI Overview

## Project Overview
ClinicFlowAC is an open healthcare workflow automation platform designed for small and mid-sized clinics.

OVWI (Open Verifiable Webhook Infrastructure) provides API verification, usage tracking, and monetization.

Live Demo:
- https://www.cyzora.com
- https://ovwi.cyzora.com

---

## Architecture

User → ClinicFlowAC → OVWI → API → Billing

---

## Current Status

- Live frontend deployed
- Functional UI and routing
- Authentication flow
- OVWI API running
- Billing integration (LemonSqueezy)
- Webhook system active

---

## Technical Stack

- Frontend: React (Vite)
- Backend: Node / Dart
- Infrastructure: OVWI
- Payments: LemonSqueezy
- Deployment: Vercel + Railway

---

## Roadmap

- Improve authentication
- Expand clinic modules
- Stabilize API infrastructure
- Enhance developer documentation

---

## Why It Matters

ClinicFlowAC provides an open alternative to proprietary healthcare systems.

OVWI introduces a reusable infrastructure for API verification and monetization.

Together, they enable a more open and developer-friendly service ecosystem.
